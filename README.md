# Todo-Api-Flask

# How to run on local machine
## Clone Repo
Open terminal

```
git clone https://github.com/nathan-1995/Todo-Api-Flask.git
```

```
cd Todo-Api-Flask
```

```
docker compose -f docker-compose.eval.yml up -d --build
```

Open web browser and go to:

```
http://localhost:3000/
```
